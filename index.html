Notes on the deliverable. Each frame above is a self-contained 1440 × 900 artboard, the standard desktop ratio used in Figma. To bring these into Figma, you can either screenshot any frame and place it on a Figma canvas at 1:1 scale, or rebuild the components from this file's clean CSS (tokens are declared as variables at the top). The grid component, clue rows, suggestion list, and modal are all built from a small, consistent token set — radius 3px, navy #000080 for accent only, warm-gray rules, and a Fraunces/Inter type pairing. The first frame is fully interactive so you can feel the placement, selection, and direction-toggle behaviors before committing to functionality work.
